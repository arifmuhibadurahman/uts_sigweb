Praktikum Sistem Informasi Bebrbasis Website

Pada Halaman ini memuat penjelasan untuk praktikum Minggu 2, 4, 5, 6, 7 :

  • Minggu 2 Pengenalan kerangka dasar website yaitu HTML, pada praktikum minggu ini terdapat 3 soal yaitu tentang pembuatan website dasar yang mengikuti gambar dari sebuah foto hasil screenshot yang sudah jadi
  • Minggu 4 Pada minggu ini diperkenalkan bahasa pemrograman css dan javascript, dengan contoh membuat program untuk Hitung Faktorial, Validasi Password, Validasi Masukan Pengguna, Menampilkan Bilangan Prima, Menampilkan Bilangan Fibonacci
  • Minggu 5 Pada minggu ini kita diperkenalkan untuk menggunakan salah satu framework dari css untuk membuat website yaitu bootstrap dan juga pengenalan mengenai terminal dari ubuntu untuk melakukan deploy melalui git yang mana lebih cepat dan tidak menimbulkan kerusakan file
  • Minggu 6 Pada minggu ini kita diperkenalkan dengan peta leaflet.js yang mana dapat diakses mealui cdn sehingga tidak perlu mendownload secara lokal
  • Minggu 7 Pada minggu ini kita diperkenalkan dengan dua macam tampilan peta yaitu dengan tilemap layer dan feature layer dengan keterangan tilemap layer adalah basemap biasa leaflet yang akan diguanakan untuk menampilkan peta, sedangkan feature layer adalah cara kita menampilkan file seperti shp dan geojson yang akan dibaca oleh website sehingga akan tampil di dalamnya Tabel berisi nim dan nama
